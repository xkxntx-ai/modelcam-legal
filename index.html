購入・サブスクリプションについて

ModelCam Pro は Apple の App Store を通じて提供されます。購入、解約、返金、請求に関する操作は Apple ID のサブスクリプション管理画面から行えます。

  • 解約: iOS の「設定」アプリ → Apple ID → サブスクリプション
  • 購入の復元: ModelCam アプリ内の設定画面、または Pro 画面の「復元」ボタン
